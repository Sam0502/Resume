This repository contains a small static site that serves as an interactive resume and portfolio for Hoe Hynn Chan. The site includes a main interactive resume (`index.html`) and a printable CV export (PDF) generated from the page.

Quick start
- Open `index.html` in a browser to view the interactive resume.
- Use the "Download PDF" button in the top-right of the page to export a clean, text-first CV as a PDF.

Files of note
- `index.html` — main interactive resume and portfolio page.
- `index.css` — styles for the interactive resume.
- `template/resume.css` — styles used by the printable CV template.
- `template/cv.html` — optional printable CV page (if present).

Customizing
- Edit `index.html` to update text, projects, or contact details.
- Replace images (if any) with local paths and update references in the header or template files.
